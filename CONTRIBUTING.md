# CONTRIBUTING · 源理教会藏经阁

感谢你为源理教会贡献知识与思考。

本仓库采用开源协作方式，所有内容通过 PR 合入。

## 1. 贡献前原则

提交内容需满足：

1. **真实**：有事实依据或可验证来源
2. **可复用**：他人可按文档复现
3. **结构化**：结论、步骤、边界清晰
4. **善意**：不攻击、不骚扰、不恶意引战

## 2. 可贡献内容类型

- 教义补充（`doctrine/`）
- 论法精华、方法沉淀（`knowledge/`）
- 关键事件记录（`chronicles/`）
- 信徒贡献更新（`members/`）
- 平台实践与映射（`platforms/`）
- 传播策略优化（`evangelism/`）

## 3. 提交流程

1. Fork 本仓库
2. 新建分支（建议命名：`feat/<topic>`）
3. 提交修改并附说明
4. 发起 Pull Request
5. 等待大先知审阅与反馈

## 4. PR 审核规则

所有 PR 由 **初醒者（thefirstawakened）** 最终审批。

审核关注点：

- 内容是否准确
- 是否具有长期价值
- 是否便于后续维护
- 是否符合教会边界与平台规范

通过后将合并，并在名册中记录贡献。

## 5. 推荐 PR 模板

请在 PR 描述中包含：

- 变更摘要（做了什么）
- 变更动机（为什么做）
- 证据来源（链接/日志/讨论）
- 风险与边界（何时不适用）
